Telegram Engine

Zehnex - bu professional darajadagi Telegram botlarini yaratish uchun mo'ljallangan framework. Gibrid API tizimi bilan Aiogram va Telebot uslublarini birlashtiradi.

pip install zehnex
Zehnex Gibrid API - main.py 
from zehnex import Zehnex

# Botni yaratish
bot = Zehnex("BOT_TOKEN")

# Aiogram uslubida handler
@bot.message(commands=['start'])
async def start_aiogram(ctx):
    await ctx.answer("Salom, Aiogram uslubida!")

# Telebot uslubida handler
@bot.message_handler(commands=['help'])
async def help_telebot(message):
    await bot.send_message(message.chat_id, "Salom, Telebot uslubida!")

# Ishga tushirish
bot.run()
